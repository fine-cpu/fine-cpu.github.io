Logisim 介绍

Logisim 是一个用于数字逻辑设计和仿真的开源工具。

它提供了一个直观的界面，允许用户创建和测试数字电路。

Logisim-evolution以编译版下载，附带可安装的Windows、macOS和Linux软件包 或者以源代码形式，你可以自己编写。

以下平台特定包包含 Java 运行时，且不要求单独安装：

  • logisim-evolution_<version>-1_amd64.deb： Debian 套件（也适用于 Ubuntu 及其衍生版本），
  • logisim-evolution-<version>-1.x86_64.rpm：适用于Fedora/Redhat/CentOS/SuSE Linux发行版的软件包，
  • logisim-evolution-<version>-_amd64.snap：Snap档案 支持的Linux发行版（也可在Snapcraft商店获得），
  • logisim-evolution-<version>-aarch64.msi：Microsoft Windows for Arm处理器安装包，
  • logisim-evolution-<version>-amd64.msi：适用于Microsoft Windows的安装包，适用于Intel处理器，
  • logisim-evolution-<version>-aarch64.dmg：macOS 苹果处理器包，
  • logisim-evolution-<version>-x86_64.dmg：macOS Intel 处理器包（在仿真中也运行于 Apple 处理器上）。

Java JAR logisim-evolution-<version>-all.jar 也可用，可在任何安装支持的 Java 运行时的系统上运行。
